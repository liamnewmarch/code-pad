# Code Pad

Code Pad is a scratchpad for HTML, CSS and JavaScript ideas which works offline.

Check it out at [code-pad.web.app](https://code-pad.web.app)

## Features

- Write HTML, CSS, and JavaScript and see the result
- Syntax highlighting and autocomplete for all three languages
- Projects are saved to the cloud and synced across devices when signed in with Google
- Fully offline-capable — install it as a PWA and use it without an internet connection

## Use cases

- Quickly prototype a UI idea without setting up a project
- Experiment with CSS layouts or JavaScript logic in isolation
- Keep a personal collection of snippets and mini-projects, accessible from any device

## Development

```bash
npm start        # Vite dev server with hot reload
npm test         # ESLint
npm run build    # Production build → dist/
npm run serve    # Firebase emulator only
npm run deploy   # Production build + deploy to Firebase Hosting
```
